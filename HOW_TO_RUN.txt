# 🎮 LINE Chat — نسخه تک‌فایلی (به‌روزرسانی شده)

همه‌چیز توی یک فایل به اسم **`run.py`** هست. فقط یه پکیج باید نصب کنی.

## ✅ مراحل اجرا

### ۱. پایتون نصب کن (اگه نداری)
https://www.python.org/downloads
(روی ویندوز موقع نصب حتما تیک **"Add Python to PATH"** رو بزن)

### ۲. ترمینال/CMD رو باز کن، بیا توی پوشه‌ای که `run.py` توشه

### ۳. پکیج لازم رو نصب کن
```
pip install aiohttp
```
اگه این خطا رو گرفتی: `error: externally-managed-environment`
به‌جاش بزن:
```
pip install aiohttp --break-system-packages
```

### ۴. سرور رو روشن کن
```
python run.py
```
(اگه کار نکرد: `python3 run.py`)

### ۵. برو تو مرورگر
```
http://localhost:8080
```

## 🔑 کد ادمین (مهم!)

برخلاف قبل، **دیگه اولین نفری که ثبت‌نام کنه خودکار ادمین نمی‌شه.** فقط کسی که کد مخصوص رو بدونه می‌تونه ادمین بشه:

- **کد پیش‌فرض:** `line-admin-2026`
- موقع ثبت‌نام یه فیلد «کد ادمین (اختیاری)» هست — این کد رو اونجا بزن تا ادمین بشی.
- اگه قبلاً بدون کد ثبت‌نام کردی، می‌تونی از پروفایلت (روی اسم خودت کلیک کن) کد رو بعداً وارد کنی.

**🔒 پیشنهاد می‌کنم این کد رو عوض کنی** تا کس دیگه‌ای نتونه ادمین بشه. دو راه داری:

**راه ۱ (ساده‌تر):** فایل `run.py` رو با Notepad یا هر ادیتور متنی باز کن، دنبال این خط بگرد:
```python
ADMIN_CODE = os.environ.get("ADMIN_CODE", "line-admin-2026")
```
و `"line-admin-2026"` رو با کد خودت عوض کن.

**راه ۲ (بدون دستکاری فایل):** قبل از اجرا کردن، این environment variable رو ست کن:
```
# ویندوز (CMD):
set ADMIN_CODE=my-secret-code
python run.py

# مک/لینوکس:
ADMIN_CODE=my-secret-code python run.py
```

## ✨ امکانات جدید این نسخه
- 📞 **ویس‌چت حالا یعنی «تماس»** — هر کاربری (نه فقط ادمین) می‌تونه از بخش «ساخت روم»، یه تماس صوتی با اسم و آیکون دلخواه بسازه. بعدش یه کد مخصوص می‌گیره که می‌تونه به دوستاش بده؛ اونا از قسمت «🔑 ورود با کد» وارد تماس می‌شن. درست مثل تماس‌های گروهی تلگرام. (دکمه‌ی میکروفون قدیمی از بالای صفحه حذف شد و به‌جاش یه صفحه‌ی تماس مخصوص با دکمه‌ی «ورود به تماس» / «پایان تماس» اضافه شده.)
- 🚫 **نشان رتبه (Noob → Legend) هم حذف شد** — قبلاً فقط لیدربورد/امتیاز حذف شده بود، الان خود نشان‌های رتبه هم از همه‌جا (پروفایل، پیام‌ها، لیست اعضا) حذف شدن.
- 🌑 **تم Telegram حالا مشکیه** — به‌جای نسخه‌ی روشن، یه تم تاریک شبیه حالت دارک تلگرام جاش گذاشته شده (همچنان پیش‌فرضه).
- ➕ **دکمه‌ی واحد «+» برای آپلود** — به‌جای دکمه‌ی جدا برای عکس، یه دکمه‌ی «+» ساده کنار باکس پیام هست؛ دکمه‌ی استیکر (🎟️) هم کنارش مونده. بقیه‌ی دکمه‌های اضافه (بولد، کد، اسپویل، ایموجی) حذف شدن تا فضای کنار باکس پیام ساده بمونه.
- 🖼️ **فرمت‌های بیشتر عکس قابل آپلودن** — png, jpg, jpeg, gif, webp, bmp, tiff, heic, heif, avif, jfif, ico (فرمت svg به‌خاطر ریسک امنیتی که می‌تونه کد مخفی توش باشه، عمداً کنار گذاشته شده).
- 🎮 **۵ روم بازی معروف با عکس آواتار مرتبط** — Valorant، Minecraft، PUBG Mobile، Free Fire، Call of Duty Mobile (ترکیبی از کامپیوتر و موبایل)؛ روم چت عمومی هم آواتار مخصوص خودش رو داره. این آیکون‌ها طراحی‌های ساده‌ی خودمونن (نه آرت/لوگوی واقعی بازی‌ها) که مشکل کپی‌رایت نداشته باشن.
- 🔐 **ساخت روم عمومی/گیمینگ فقط با ادمین، ولی تماس صوتی و روم خصوصی برای همه** — هر کاربری می‌تونه روم خصوصی یا تماس صوتی بسازه؛ فقط روم‌های «رسمی» (عمومی/گیمینگ) دست ادمینه.
- 🔒 **روم‌های خصوصی و تماس‌های صوتی مثل تلگرام** — توی لیست عمومی دیده نمی‌شن؛ فقط با کد دعوت می‌شه واردشون شد.
- ⚙️ **مدیریت روم برای سازنده‌ش** — اسم/آیکون رو عوض کنه، کد دعوت رو عوض کنه، یا کل روم رو حذف کنه.
- 🗑️ **پیام‌ها بعد از ۲۴ ساعت خودکار حذف می‌شن**
- 🔑 **پنل مدیریت فقط با کد مخصوص**
